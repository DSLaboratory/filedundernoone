# FILED UNDER ~~NO ONE~~

**The landing site for the novel and its companion stealth game.**
Live at **[filedundernoone.uk](https://filedundernoone.uk)**.

---

> She kept the country's secrets. The country kept one — about her.

For eleven years, Margaret Vane sat at the Registry and decided what the country
was allowed to see. Then she opened a ledger she was meant to file and not read —
and found it was quietly killing the living by writing them down as dead. She did
the correct thing. She reported it. By morning she had never existed.

The book stops at the threshold. The **companion game** is the night she goes back
in — a free, top-down stealth game in the same dark.

## What's in here

This repository is **only the public landing page** — a single, self-contained
`index.html` with its fonts and imagery. No tracking, no analytics, no build step,
no runtime dependencies. It is served as a static site by GitHub Pages.

- `index.html` — the page (hand-written HTML/CSS, zero dependencies)
- `fonts/` — Bricolage Grotesque + Spline Sans Mono (OFL), self-hosted woff2
- `img/` — the dossier imagery
- `CNAME` — the custom domain
